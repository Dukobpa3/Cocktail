======================
== TEST PROJECT ==
======================

A project template, ready to build to for small html/css/haxeJS tests

use src/Main.hx for your test code, src/index.html for yout test html and bin/style.css for styling,

then build with the hxml files in this folder

You can then run yout test in flash or js by opening the html files in the bin folder in a browser
